# 插件简介

sysSentry巡检框架可对巡检能力进行管理，所有巡检能力均以插件的形式存在，不同插件支持不同巡检能力。sysSentry目前提供的插件如下：

- 慢IO插件：进行慢盘故障检测。当巡检插件检测到系统存在慢盘时，会将结构上报给xalarmd服务，用户可通过注册告警或get\_alarm命令的方式查看告警结果。

- 灵衢可靠性插件，实现对panic/reboot/bmc下电/oom/ub内存故障等事件的监听上报功能。

- bmc ras告警上报插件， 实现获取bmc上各种ras告警信息，当插件检测到bmc ras告警时，会将结果上报给xalarmd服务。

- CPU故障巡检插件，实现CPU在线巡检和故障核隔离功能。
